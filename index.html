Politique de Confidentialité

Dernière mise à jour : 21 octobre 2025

1. Introduction

Cette application ("Bot Facebook") est un outil automatisé de publication de contenu éducatif sur Facebook. Nous nous engageons à protéger votre vie privée.

2. Données collectées

Notre application collecte uniquement :

  • Les informations publiques de votre Page Facebook (nom, ID)
  • Les statistiques publiques des publications (likes, commentaires, partages)
  • Aucune donnée personnelle des utilisateurs

3. Utilisation des données

Les données collectées sont utilisées uniquement pour :

  • Publier du contenu sur votre Page Facebook
  • Analyser les performances des publications
  • Améliorer le contenu généré automatiquement

4. Partage des données

Nous ne partageons, ne vendons ni ne louons vos données à des tiers. Les données restent entre vous et Facebook.

5. Sécurité

Vos tokens d'accès sont stockés de manière sécurisée et ne sont jamais exposés publiquement.

6. Vos droits

Vous pouvez à tout moment :

  • Révoquer l'accès de l'application à votre Page Facebook
  • Supprimer les données stockées localement
  • Nous contacter pour toute question

7. APIs tierces utilisées

Cette application utilise :

  • Facebook Graph API - Pour publier du contenu
  • OpenRouter API - Pour générer du texte
  • Hugging Face API - Pour générer des images

Chacune de ces APIs a sa propre politique de confidentialité.

8. Modifications

Nous nous réservons le droit de modifier cette politique à tout moment. Les modifications seront publiées sur cette page.
